# 《个性化学习启动蓝图》

## 目标：最快速度掌握非凸科技 Rust 量化交易系统实习所需技术栈，并完成一个匹配岗位的项目

### 非凸科技
Rust 开发工程师实习岗位要求： 
1.设计并开发基于Rust的高性能，低时延算法交易系统。 
2.设计并开发数据处理平台，监控运维平台。 
3.设计并开发面向客户的高可用交易工具等。 
4.设计并开发策略相关的回测平台。

你的目标不应是“学完 Rust”，而应是：
> **用 Rust 做出一个事件驱动的交易系统 mini 版：能接收模拟行情、维护订单簿、运行策略、生成订单、模拟成交、做回测、暴露 API、记录日志和性能指标。**

这比泛学 Rust 更适合这个岗位。

# 阶段一：战略规划与研究

## 第 1 步：了解全局 The Big Picture
这个岗位的核心不是单纯写 Rust 语法题，而是用 Rust 构建 **高性能、低延迟、可靠、可观测的交易基础设施** 。它要解决的问题是：行情数据高速进入系统后，系统如何快速解析、分发、驱动策略决策、管理订单、模拟或连接交易执行，并且能对策略进行历史回测、监控和运维。Rust 的价值在于它适合性能敏感、并发密集、可靠性要求高的系统开发；Tokio 则提供异步运行时、网络、调度、定时器等基础能力，常用于 Rust 网络服务。([Rust 文档](https://doc.rust-lang.org/book/?utm_source=chatgpt.com "The Rust Programming Language"))
这个岗位涉及 5 个主要子方向：
1. **Rust 系统编程基础**：所有权、借用、生命周期、错误处理、trait、泛型、并发。
2. **异步网络与低延迟服务** ：Tokio、TCP/WebSocket、消息分发、channel、超时、重连、背压。
3. **交易系统核心模块** ：行情、订单簿、策略接口、订单管理、撮合模拟、风险控制。
4. **回测与数据处理平台** ：CSV/Parquet、时间序列事件回放、收益统计、Polars/Arrow。
5. **平台化与可观测性** ：Axum API、SQLx 存储、tracing 日志、Prometheus 风格指标、benchmark。

## 第 2 步：确定范围 Define the Scope
### 本次学习的聚焦范围
不是：
> “全面掌握 Rust 技术栈。”

而是：
> **掌握 Rust + Tokio + Axum + Serde + SQLx + tracing + Criterion，并用它们完成一个事件驱动交易回测与订单管理系统。**

项目最终形态：
```text
market-data-simulator
↓
event-bus / channel
↓
order-book
↓
strategy-engine
↓
order-manager
↓
execution-simulator
↓
backtest-report + API + metrics
```
### 暂时划出范围外的内容
为了最快应聘，以下内容暂时不深究：

|暂不深究|原因|
|---|---|
|unsafe Rust 深度优化|实习阶段容易走偏，先写安全、正确、可解释的代码|
|Rust 编译器源码|与岗位产出距离太远|
|过程宏|加分项，但不是交易系统核心|
|WebAssembly|与该岗位相关度低|
|GUI 桌面开发|客户工具更可能是 API / Web 服务后端|
|真实交易所接入|风险高、复杂度高，先做模拟网关|
|复杂量化策略|你应聘的是系统开发，不是策略研究|
|高频交易硬件优化|如 DPDK、kernel bypass、CPU pinning，先了解概念即可|
|分布式大数据平台|先做单机数据处理与回测闭环|

### 20/80 核心
你只需要优先掌握这 20%：
1. Rust 所有权、结构体、枚举、trait、错误处理。
2. Tokio 异步任务、channel、TCP/WebSocket 基础。
3. 订单簿、事件驱动架构、策略 trait。
4. Axum API + Serde JSON。
5. tracing 日志 + Criterion benchmark。
6. CSV 数据读取、回测报告输出。

# 最关键的学习纪律
你现在刚学 Rust，最容易犯的错误是：
> 学太多资料，写太少代码。

所以你的规则是：
```text
每读 30 分钟资料，必须写 60 分钟代码。
每实现一个模块，必须写测试。
每遇到编译错误，不要绕开，必须解释清楚。
每三天更新一次 README。
每周至少产出一个可运行版本。
```
你的路线不是“先学完再做项目”，而是：
> **用项目倒逼学习 Rust。**

这条路线最贴合该岗位，也最快形成面试材料。
